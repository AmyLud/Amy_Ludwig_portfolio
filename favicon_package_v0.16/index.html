Just my type

HTML | JQuery | SASS | Design

Just-my-Type is a typewriter simulator with a few extras. There's no backspace (ink can't erase!), depleting ink, and an adjustable "jankiness" factor. Ever feel like going back to before computers were everywhere? Me neither, but use Just-My-Type for 10 minutes and you'll definitely never take your word processor for granted again.

Halpq

React | SASS | Design | Team Programming

HALPQ is a question asking platform with in-classroom chat functionality. Students are able to ask questions in their respective classes, and Admins are able to respond to questions, mark them as resolved, and manage user priveledges. The solution to using 3rd-party platforms, HALPQ was built with React and uses Firebase to store data.
